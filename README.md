# PDK_Sample_Program

이 리포지토리는 PDK 환경에서 단일 및 다중 디바이스(Multi-Device)를 연결하고 제어하기 위한 C# 및 C++ 샘플 프로그램과 가이드 문서를 제공합니다.

## 📁 프로젝트 구조

* **`C#/PDK_Motion_Example/`** * C# 환경에서 PDK 모션 제어를 테스트해 볼 수 있는 샘플 예제입니다.
* **`C++/CommunicationLibraryDemoVC/`**
  * C++(VC) 환경에서 디바이스와의 통신 라이브러리를 활용하는 데모 코드입니다.
* **`PMAC Project/PDK_Sample_for_PMAC/`**
  * PMAC 장비를 위한 PDK 샘플 프로젝트 파일입니다.
* **`multi_device_connection_guide.html`**
  * 다중 디바이스(Multi-Device) 연결 시 C# 및 C++ 환경에서의 주의사항과 상세 연결 가이드를 담은 웹 문서입니다.

## ✨ 주요 특징 및 업데이트 사항

* **다중 디바이스 연결 지원:** 단일 연결뿐만 아니라 여러 대의 디바이스를 동시에 연결하고 제어하기 위한 코드가 적용되어 있습니다. (v1.0.0 릴리스 참고)
* **상대 경로 이슈 해결:** 기존 소스 코드의 불안정한 상대 경로 참조를 **PDK 설치 경로** 기준으로 수정하여, 클론 후 별도의 복잡한 경로 설정 없이 바로 빌드할 수 있도록 개선되었습니다.
* **상세한 가이드 제공:** HTML 형태의 가이드 문서를 통해 개발자가 다중 디바이스 환경을 쉽게 구축할 수 있도록 돕습니다.

## 🚀 시작하기

1. 이 리포지토리를 클론(Clone)하거나 가장 최신 버전의 [Release](https://github.com/minimin333/PDK_Sample_Program/releases) 파일을 다운로드합니다.
2. 여러 대의 디바이스를 연결할 계획이라면, 코드를 열기 전에 최상위 경로에 있는 `multi_device_connection_guide.html`을 브라우저로 열어 가이드라인을 먼저 숙지해 주세요.
3. 사용하고자 하는 개발 언어(C# 또는 C++) 폴더로 진입하여 프로젝트/솔루션 파일을 열고 빌드를 진행합니다.

## 📌 주의 및 참고 사항

* 이 샘플 코드를 정상적으로 빌드하고 실행하기 위해서는 PC에 PDK가 기본 경로로 설치되어 있어야 합니다.
